# Citadel v2
## Quickstart

- Show all recipes
  ```bash
  cit
  ```
- Scaffold package/app templates
  ```bash
  cit scaffold::python
  cit scaffold::react
  cit scaffold::typescript
  ```
- Run stuff
    ```bash
    cit run::ocr
    cit run::ocr-kill
    ```
- Open repositories (generally in vscode)
  ```bash
  cit open::cit
  ```
- Add stuff to current project
  ```bash
  cit add::images
  cit add::python
  ```
